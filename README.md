# L298-Motor-Driver
This project focuses on the design of a DC motor driver PCB using the L298N motor driver IC.  The schematic and PCB layout were developed in KiCad with proper component placement, routing, and net labeling.  The design was validated using ERC and DRC checks and is ready for fabrication.

# MotorLink – DC Motor Driver PCB Design

## Description
This project involves the design of a DC motor driver PCB using the L298N motor driver IC. 
The schematic and PCB layout were developed using KiCad.

## Features
- Forward and reverse motor control
- Proper net labeling and routing
- Compact PCB layout

## Tools Used
- KiCad

## Design Validation
- Performed ERC (Electrical Rule Check)
- Performed DRC (Design Rule Check)

## Files Included
- Schematic design files
- PCB layout files
- (Optional) Gerber files for fabrication
